# Cleanex

[Cleanex](https://hex.pm/packages/cleanex) is a Data cleansing & aggregating libraries in Elixir. Here is an example:

```elixir
iex> 

```

See the [online documentation](https://hexdocs.pm/cleanex).

## Installation

Add to your ```mix.exs``` file:

```elixir
def deps do
  [
    { :cleanex, "~> 0.0.1" }
  ]
end
```
